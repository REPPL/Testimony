# Development record

The durable record for this repository: design briefs, dated plans and
research notes, and architecture decision records (MADR format, sequential
`NNNN`, under `decisions/adrs/` once the first one lands).

Subdirectories are created only when there is content to put in them.
Session-level decisions live in [`../work/DECISIONS.md`](../work/DECISIONS.md);
architecture-shaping ones graduate here.
