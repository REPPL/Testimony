# Meta

Conventions for this brief:

- **Numbered folders and files** with two-digit zero-padded prefixes (`01-`,
  `02-`, …) so sort order survives past nine entries; **kebab-case** names
  throughout, matching the rest of the project.
- **`README.md` is the index** and deliberately carries no prefix — it is the
  entry point, not a section.
- The brief describes the **current state only**. A section with nothing real
  to say is omitted rather than padded; gaps in the numbering are deliberate.
- Where the brief and the code disagree, the code wins and the brief is the
  bug. The original design note is preserved unchanged in
  [`../research/2026-07-17-architecture-note.md`](../research/2026-07-17-architecture-note.md).
